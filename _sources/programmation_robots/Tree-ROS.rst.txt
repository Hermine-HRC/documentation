Programmation ROS
=================

Ici est détaillé tout ce qui concerne le code ROS du robot (Herminebot).

Plusieurs choses ont été mises en place afin d'avoir un code de qualité. Est notamment inclus l'exécution automatique
de tests (dont linter) couplé avec un *code coverage* afin de connaitre quelles parties du code sont testées ou non.
